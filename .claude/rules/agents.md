## Available Agents

Specialized agents in `.claude/agents/` can be invoked via the Agent tool for domain-specific tasks:

- **bitcoin-expert** — Bitcoin protocol, consensus rules, cryptography, BSV-specific features
- **test-writer-fixer** — Writes and fixes tests after code changes
- **api-tester** — API load testing and contract validation
- **backend-architect** — System design and architecture decisions
- **document-reviewer** — Documentation quality review and accuracy audit

Vendored from [VoltAgent/awesome-claude-code-subagents](https://github.com/VoltAgent/awesome-claude-code-subagents):

- **golang-pro** — Go concurrency, performance, idiomatic patterns, error handling
- **code-reviewer** — Language-agnostic code review (quality, security, best practices)
- **security-auditor** — Security audit: consensus/UTXO/crypto, vulnerability analysis
- **qa-expert** — Test strategy, coverage, race/fuzz, CI quality gates
- **performance-engineer** — Hot-path profiling, lock contention, DB/throughput tuning
- **typescript-pro** — TypeScript correctness, advanced types, end-to-end type safety
- **frontend-developer** — Frontend/dashboard UI (the `ui/dashboard` Svelte app)
- **security-engineer** — Infra/CI/supply-chain security, pipeline hardening
- **penetration-tester** — Offensive security: DoS vectors, panic chains, exploitation
