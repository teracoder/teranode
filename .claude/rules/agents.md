## Available Agents

Specialized agents in `.claude/agents/` can be invoked via the Agent tool for domain-specific tasks:

- **bitcoin-expert** — Bitcoin protocol, consensus rules, cryptography, BSV-specific features
- **test-writer-fixer** — Writes and fixes tests after code changes
- **api-tester** — API load testing and contract validation
- **backend-architect** — System design and architecture decisions
- **document-reviewer** — Documentation quality review and accuracy audit
- **review-docs** — Documentation review workflow
